# ci-artifact-node

## Overview

This repository functions as a **personal CI artifact node**.

It automates build workflows and produces release artifacts using
GitHub Actions and upstream open-source tooling.

The generated artifacts are publicly accessible due to GitHub
release mechanics. This does **not** imply endorsement, distribution,
or support intent.

---

## Scope

This repository:

- Contains automation workflows only
- Does not include upstream project source code
- Does not operate as a product or service
- Exists primarily for internal and personal automation

Repository structure, workflows, and outputs may change at any time.

---

## Public Artifacts Notice

Release artifacts are publicly downloadable.

This repository does **not**:
- Promote the artifacts
- Provide user support
- Offer guarantees of functionality, safety, or updates

Public availability exists solely as a technical requirement of the
CI and release system in use.
