# ci-artifact-node

## Overview

This repository functions as a **personal CI artifact node**.

It exists solely to automate build workflows and generate release
artifacts using GitHub Actions and upstream open-source tooling.

All generated artifacts are publicly accessible due to GitHub
release mechanics. Public availability does **not** imply endorsement,
distribution intent, promotion, or support.

This repository:

- Contains automation workflows only
- Does **not** include upstream project source code
- Does **not** operate as a product, service, or platform
- Exists strictly for internal and personal automation purposes

break, or be removed at any time without notice.
